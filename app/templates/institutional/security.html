Data Protection

Encryption

  • All data encrypted in transit using TLS 1.3
  • Data encrypted at rest using AES-256
  • Database connections encrypted end-to-end
  • Passwords hashed using bcrypt with salting

Infrastructure

  • Hosted on enterprise-grade cloud infrastructure (AWS/Heroku)
  • Automatic daily backups with 30-day retention
  • Geographic redundancy for disaster recovery
  • 99.9% uptime SLA for enterprise customers

Access Control

  • Role-based access control (RBAC)
  • SSO/SAML integration for institutional authentication
  • Multi-factor authentication available
  • Session timeout and secure cookie policies
  • Rate limiting on all API endpoints

AI & Data Privacy

Our AI features are designed with privacy in mind:

  • AI processing uses industry-standard models with data protection agreements
  • Student content is never used to train AI models
  • AI requests are processed in real-time and not stored long-term
  • Institutions can disable AI features if required by policy
  • All AI interactions are logged for audit purposes

Security Practices

Application Security

  • CSRF protection on all forms
  • SQL injection prevention via parameterized queries
  • XSS protection with content sanitization
  • Security headers (HSTS, CSP, X-Frame-Options)
  • Regular dependency vulnerability scanning

Monitoring & Incident Response

  • 24/7 automated security monitoring
  • Real-time alerting for suspicious activity
  • Documented incident response procedures
  • Security team on-call for emergencies

Data Ownership & Portability

  • Your institution owns all data created by your users
  • Full data export available in standard formats (JSON, CSV)
  • Data deletion upon contract termination per your requirements
  • No vendor lock-in - export your data anytime
